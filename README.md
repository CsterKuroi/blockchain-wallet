# unichain-client

### Quick Start

```
sudo pip3 install -r requirements.txt
python3 manage.py
```

Then, open `localhost:500` in your browser

### Default Config (.unichain-account)

```
{
    "server": {
        "port": "9984",
        "host": "localhost"
    },
    "username": "user",
    "keypair": {
```
